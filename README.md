# PROS Worship Website

A beautiful, theology-focused website for Public Reading of Scripture (PROS) worship.

## About PROS Worship

PROS worship centers on the conviction that **God speaks through His Word**. When Scripture is read aloud, God addresses His people—shaping faith, forming the church, and directing our lives.

> "Devote yourself to the public reading of Scripture, to preaching and to teaching." — 1 Timothy 4:13

## Website Features

- 📖 **Biblical Foundation** - Comprehensive Scripture references
- 🏛️ **Early Church Quotes** - Justin Martyr, Tertullian, Augustine
- 🎨 **Beautiful Design** - Warm, inviting aesthetics
- 📱 **Responsive** - Works on all devices
- ⚡ **Fast Loading** - Pure HTML/CSS, no JavaScript dependencies

## Content Sections

1. **Welcome** - Invitation to listen and hear God's Word
2. **Why PROS?** - Theological foundation
3. **Biblical Pattern** - Scripture references from Moses to Paul
4. **Early Church** - Historic practice of the first Christians
5. **Why It Matters** - Contemporary relevance

## Technical

- Built with semantic HTML5
- Custom CSS with CSS variables for theming
- Mobile-first responsive design
- Accessible WCAG compliance

## Deployment

This site is designed to be hosted on GitHub Pages or any static hosting service.
